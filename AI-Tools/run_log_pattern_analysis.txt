Type ESQL

Query

FROM ste-logs-*
  | SAMPLE ?sampleSize
  | STATS Count=COUNT(*)/?sampleSize BY Pattern=CATEGORIZE(log.message)
  | SORT Count DESC
  | LIMIT 100

ESQL Parameters
-------------------------------
sampleSize
The sample size of the pattern analysis default to .001
